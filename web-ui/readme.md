# Asciimaton web interface

Made with Reactjs.

## Dev :

- `yarn` or `npm install`
- `npm start` 
- serves on `localhost:8080` with hot reloading

## Build :

- `yarn` or `npm install`
- `npm run build`

## Test

Test socketio server with `node ./test/node-socketio-server.js`.
It will instanciate a socket on `localhost:4200`.
